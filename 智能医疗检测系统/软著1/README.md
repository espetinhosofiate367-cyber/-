## 🎯 项目简介

**动态逐帧检测系统** 是一套基于Python开发的高精度医学图像分析平台，专门用于动态结节检测、实时数据分析和智能诊断辅助。系统集成了先进的机器学习算法、实时可视化技术和全面的统计分析功能，为生物医学工程和医学信息工程领域提供专业的解决方案。

### 🏥 应用领域
- **医学影像分析**: 结节检测、病灶识别、动态监测
- **生物医学研究**: 数据分析、模式识别、统计建模
- **智能医疗设备**: 实时检测、自动诊断、质量控制
- **临床辅助诊断**: 风险评估、趋势分析、报告生成

---

## ✨ 核心特性

### 🔬 高精度检测算法
- **高斯混合模型 (GMM)**: 先进的概率建模技术
- **形态学后处理**: 优化检测结果，减少假阳性
- **多维特征融合**: 面积、圆度、强度、位置综合分析
- **自适应阈值**: 根据数据特征动态调整检测参数

### 📊 实时可视化系统
- **动态图表更新**: 实时显示检测结果和统计数据
- **多视图同步**: 原始数据、检测结果、趋势分析同步展示
- **3D立体显示**: 三维数据分布和变化趋势可视化
- **交互式界面**: 参数实时调整，结果即时反馈

### 📈 智能统计分析
- **趋势分析**: 线性回归、平滑滤波、显著性检验
- **异常检测**: Z-score、IQR、孤立森林多算法融合
- **周期性分析**: 峰值检测、间隔统计、规律性评估
- **相关性挖掘**: 特征间关系量化和可视化

### 🚀 高性能处理
- **多线程架构**: 并行处理，提升系统响应速度
- **批量处理**: 支持大规模数据集自动化分析
- **内存优化**: 高效的数据结构和算法实现
- **实时通信**: 串口数据采集和设备集成

## 🚀 快速开始

### 1. 环境要求
- **Python**: 3.8 - 3.11 (推荐 3.9)
- **操作系统**: Windows 10/11, macOS 10.14+, Linux Ubuntu 18.04+
- **内存**: 8GB RAM (推荐 16GB)
- **存储**: 2GB 可用空间

### 2. 安装步骤

```bash
# 1. 克隆或下载项目文件
# 确保所有源代码文件在同一目录

# 2. 创建虚拟环境 (推荐)
python -m venv detection_env

# 3. 激活虚拟环境
# Windows:
detection_env\Scripts\activate
# macOS/Linux:
source detection_env/bin/activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 验证安装
python -c "import numpy, pandas, matplotlib, sklearn; print('安装成功!')"
```

### 3. 启动系统

```bash
# 启动主程序 (推荐)
python main_detection_app.py

# 或直接启动检测界面
python modern_detection_gui.py

# 或运行统计分析模块
python 统计.py
```
